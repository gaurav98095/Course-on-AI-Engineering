# Checklist

- Visual purpose is clear.
- Mermaid or SVG description is included for concepts with structure.
- 2–4 real photos are curated for the lesson's story beats.
- Every photo's license is CC BY, CC BY-SA, CC0, or public domain — verified, not assumed.
- Photos are downloaded to `docs/assets/images/` (no hotlinks) and resized to ≤ 1400px.
- Every figure has a caption that teaches something and credits source + license.
- Headshots use the `portrait` class.
- No visual is added just for decoration.
