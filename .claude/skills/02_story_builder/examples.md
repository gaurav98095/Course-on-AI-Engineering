# Examples

## Example 1

Topic: KV cache

Story: A student solving a long multiplication redoes the whole sum from scratch for every new digit — until they learn to write down partial results. The lesson opens with that wasted effort, then maps it to attention recomputing keys and values for every generated token.

## Example 2

Topic: Roofline model

Story: A world-class chef stands idle in a kitchen because ingredients arrive on a slow conveyor belt. Hiring a faster chef changes nothing. That bottleneck flip — compute waiting on memory — motivates the roofline model.

## Example 3

Topic: Continuous batching

Story: A restaurant that seats a new party only when an entire section leaves has empty chairs everywhere. One that seats guests the moment any chair frees up serves twice the covers with the same tables. That is static vs continuous batching.
