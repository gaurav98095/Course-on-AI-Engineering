# Workflow: Update a Lesson

Use this workflow to revise an existing lesson while preserving style consistency.

## Procedure

1. Identify the requested change scope.
2. Inspect the affected sections only.
3. Regenerate only the impacted stage outputs.
4. Recompose the article.
5. Run `08_editor` on the updated draft.

## Preservation rules

- Keep the lesson's primary mental model unless the user asks to change it.
- Preserve notation unless the math is corrected.
- Preserve the article structure unless a structural fix is required.

## Output

- Updated lesson Markdown
- Short change log
