# Workflow: Review Existing Lesson

Use this workflow to review a completed or in-progress lesson.

## Pipeline

1. Read the lesson and its stated objective.
2. Run `08_editor` against the draft.
3. Classify findings by severity.
4. Return a focused fix list.

## Review rules

- Do not rewrite the full article unless explicitly asked.
- Do not re-plan the course unless the lesson is structurally broken.
- Focus on consistency, correctness, readability, and missing pedagogy.

## Output

- Summary
- Findings
- Suggested fixes
- Residual risks
