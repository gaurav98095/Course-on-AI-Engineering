# Workflow: Expand an Outline

Use this workflow to expand a short outline into a complete lecture.

## Pipeline

1. Convert the outline into a lesson blueprint if needed.
2. Write the lesson story.
3. Define one mental model.
4. Build the example bank (`09_example_generator`).
5. Draft the math section.
6. Draft the implementation section.
7. Draft visuals.
8. Compose the article.
9. Edit the draft.
10. Publish via `workflow_publish.md`.

## Rule

- Preserve the outline's intent.
- Add depth, examples, and transitions without changing the conceptual target.

## Output

- A single publish-ready lesson in Markdown.
